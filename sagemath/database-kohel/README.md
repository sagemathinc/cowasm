# Kohel Database For SageMath

This package provides SageMath's optional `database_kohel` data package for
CoWasm. Sage uses it for modular polynomials, Atkin polynomials, eta
polynomials, and Hilbert class polynomials.

CoWasm installs the bzip2-compressed `.dbz` tables under
`dist/wasm/share/database_kohel`.
