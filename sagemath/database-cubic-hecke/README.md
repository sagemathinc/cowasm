# Cubic Hecke Database For SageMath

This package provides SageMath's `database_cubic_hecke` package for CoWasm.
It contains precomputed representation data for cubic Hecke algebras, including
Ivan Marin's representation data and Markov trace coefficients.

CoWasm installs the importable Python package under
`dist/wasm/database_cubic_hecke`.
