# PALP

This package builds PALP lattice polytope tools used by SageMath's toric
geometry functionality.

The standalone smoke builds the 4-dimensional PALP executables with WASI
compatibility shims, installs representative test inputs, and runs `poly-4d.x`,
`class-4d.x`, `cws-4d.x`, `nef-4d.x`, and `mori-4d.x` under the WASI runner.
